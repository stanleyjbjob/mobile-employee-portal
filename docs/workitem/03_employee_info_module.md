# 03_個人資訊管理模組

## 目標
提供員工查詢與維護個人資訊、薪資單、扣繳憑單等功能，支援下載與安全存取。

## 作業內容
- 後端：
  - 設計 Employee、Payroll、TaxDocument 等資料模型
  - 實作查詢/下載 API（RESTful）
  - 權限驗證與日誌記錄
  - 撰寫單元測試
- 前端/行動端：
  - 實作個人資訊、薪資單、扣繳憑單查詢與下載 UI
  - 表單驗證與錯誤提示
  - loading 狀態處理

## 產出物
- 員工個人資訊查詢/下載功能
- 前後端單元測試

## 注意事項
- 檔案下載需考慮安全性
- 欄位驗證需前後端一致
